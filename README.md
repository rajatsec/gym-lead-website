# 🏋️‍♂️ Titan Fitness | Gym Lead-Gen Website

A modern, high-conversion landing page designed for fitness centers and personal trainers to capture leads and convert visitors into active members.

---

## 🚀 Overview
In the competitive fitness industry, speed and conversion are everything. This project focuses on a **Mobile-First** approach, delivering a 95+ PageSpeed score and a seamless WhatsApp-based lead capture system that connects clients directly to gym owners.

---

## ✨ Features
- ✅ **High-Conversion UI:** Clear Call-to-Action (CTA) buttons and pricing tables.
- 📱 **Responsive Design:** Optimized for all screen sizes (300px to 4K).
- 💬 **WhatsApp Lead Capture:** Instant communication via custom API integration.
- ⚡ **Performance Optimized:** No heavy libraries, lazy-loaded images (WebP).
- 🎨 **Modern Aesthetics:** Dark theme with high-contrast red accents (High-energy UI).

---

## 🛠️ Tech Stack
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Production Build)
- **Runtime:** Vanilla JavaScript (ES6+)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Deployment:** Vercel / Netlify

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajatsec/gym-lead-website.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
